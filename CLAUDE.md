# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is **not a software project**. This is an academic policy research repository containing documentation related to Student Evaluations of Teaching (SET) reform at Cal Poly's Orfalea College of Business (OCOB).

## Repository Structure

- **01_Proposed_Reforms/**: Core policy reform documents including existing policy analysis, recommendations, resolutions, FAQ responses, and presentations
- **02_Key_References/**: Research literature references on SET bias, gender bias in evaluations, and relevant legal cases (RFA v. Ryerson)
- **03_Training_Template/**: Training materials for performance evaluation meetings

## Content Focus

The repository addresses:
- Bias in student evaluations of teaching
- Gender bias in academic evaluations
- Policy recommendations for SET reform
- Legal precedents (Ryerson arbitration case)
